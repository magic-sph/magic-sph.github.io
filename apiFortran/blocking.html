Module containing blocking information

Needed modules

  • mem_alloc (memwrite(), bytes_allocated()): This little module is used to estimate the global memory allocation used in MagIC
  • useful (abortrun(), logwrite()): library with several useful subroutines
  • output_data (n_log_file(), log_file()): This module contains the parameters for output control
  • precision_mod: This module controls the precision used in MagIC
  • parallel_mod (n_procs(), rank_with_l1m0(), nthreads(), rank(), nlmbs_per_rank()): This module contains the blocking information
  • logic (l_finite_diff(), l_save_out()): Module containing the logicals that control the run
  • lmmapping (mappings(), deallocate_mappings(), subblocks_mappings(), deallocate_subblocks_mappings(), allocate_mappings(), allocate_subblocks_mappings())
  • constants (one()): module containing constants and parameters used in the code.
  • truncation (l_max(), minc(), lmp_max(), l_axi(), n_r_max(), m_max(), lm_max(), nrp(), n_theta_max()): This module defines the grid points and the truncation
