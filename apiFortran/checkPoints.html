Reading and storing check points (restart files)¶

readCheckPoints.f90¶

Description

This module contains the functions that can help reading and mapping of the restart files

Quick access

Description

This module contains several subroutines that can be used to store the checkpoint_#.tag files

Quick access

Needed modules

  • lmloop_data (ulmmag(), llmmag(), llm(), ulm())

subroutine storecheckpoints/store(time, dt, dtnew, n_time_step, l_stop_time, l_new_rst_file, l_ave_file, w, z, p, s, xi, b, aj, b_ic, aj_ic, dwdtlast, dzdtlast, dpdtlast, dsdtlast, dxidtlast, dbdtlast, djdtlast, dbdt_iclast, djdt_iclast)¶
This subroutine stores the results in a checkpoint file. In addition to the magnetic field and velocity potentials we also store the time derivative terms djdt(lm,nR),dbdt(lm,nR), … to allow to restart with 2nd order Adams-Bashforth scheme. To minimize the memory imprint, a gater/write strategy has been adopted here. This implies that only one global array dimension(lm_max,n_r_max) is required.
